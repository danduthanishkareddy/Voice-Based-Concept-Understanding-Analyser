# Voice-Based Concept Understanding Analyser

## Project Overview
The Voice-Based Concept Understanding Analyser is an AI-powered application that evaluates a user's conceptual understanding from spoken audio. The application converts speech to text, analyzes the response, measures similarity with reference content, evaluates audio features, detects filler words, and generates a detailed PDF report.

## Features
- Speech-to-text conversion using OpenAI Whisper
- Concept similarity analysis
- Audio feature extraction
- Filler word detection
- Performance scoring
- Automatic PDF report generation
- Streamlit-based user interface

## Technologies Used
- Python
- Streamlit
- OpenAI Whisper
- Sentence Transformers
- Librosa
- ReportLab

## How to Run
1. Clone the repository.
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Start the application:
   ```
   streamlit run app.py
   ```
4. Upload an audio file and view the generated analysis report.
